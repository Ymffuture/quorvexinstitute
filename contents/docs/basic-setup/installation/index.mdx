## HTML & CSS at Quorvex Institute

At **Quorvex Institute**, we build modern, responsive, and maintainable websites using a combination of powerful frontend technologies:

- **TailwindCSS** — A utility-first CSS framework enabling rapid, customizable styling without writing custom CSS.
- **Bulma & Bootstrap** — Two popular CSS frameworks providing grid systems and ready-to-use UI components for faster development.
- **HTML5 & CSS3** — The backbone of any web page, structured semantically and styled for accessibility and performance.

Our projects use these tools in tandem with **React.js** and **Next.js** to deliver fast, scalable web apps that look great on any device.

---

### Why TailwindCSS?

TailwindCSS lets us compose styles by applying utility classes directly in markup, dramatically speeding up design iteration and ensuring consistent UI without bulky CSS files.

**Example: A responsive card using TailwindCSS**

```jsx
export default function InfoCard() {
  return (
    <div className="max-w-sm rounded overflow-hidden shadow-lg p-6 bg-white dark:bg-gray-800">
      <h2 className="font-bold text-xl mb-2 text-gray-900 dark:text-white">Welcome to Quorvex Institute</h2>
      <p className="text-gray-700 dark:text-gray-300 text-base">
        We combine the power of React, Next.js, and modern CSS frameworks to create performant and beautiful websites.
      </p>
      <button className="mt-4 px-4 py-2 bg-blue-600 text-white rounded hover:bg-blue-700 transition">
        Learn More
      </button>
    </div>
  )
}
````

### Using Bulma and Bootstrap

While TailwindCSS is our primary styling approach, Bulma and Bootstrap come in handy for quick prototyping and access to ready-made components like navbars, modals, and forms.

**Example: A simple Bulma navbar**

```jsx
export default function BulmaNavbar() {
  return (
    <nav className="navbar is-primary" role="navigation" aria-label="main navigation">
      <div className="navbar-brand">
        <a className="navbar-item" href="/">
          <strong>Quorvex Institute</strong>
        </a>
      </div>
      <div className="navbar-menu">
        <div className="navbar-start">
          <a className="navbar-item" href="/about">
            About
          </a>
          <a className="navbar-item" href="/courses">
            Courses
          </a>
        </div>
      </div>
    </nav>
  )
}
```

**Example: Bootstrap button in React**

```jsx
import 'bootstrap/dist/css/bootstrap.min.css';

export default function BootstrapButton() {
  return (
    <button type="button" className="btn btn-success">
      Enroll Now
    </button>
  )
}
```

### Integrating with React and Next.js

By combining these CSS tools with React and Next.js, we enjoy:

* Component-driven UI with encapsulated styles
* Server-side rendering (SSR) for SEO and performance
* Static site generation (SSG) for fast loading
* Tailwind’s JIT compiler for rapid style updates
